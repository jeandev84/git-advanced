# GIT RESET (Cancel Index files)
(Вернуть предыдующее состояние все версии или определанные файлы

$ git reset

(Вернуть предыдующее состояние посление изменения)
(Мы говорим о том что мы вернем предыдующее состояние файл test.txt,
перед тем сделали коммит)
HEAD - это последние состояние ,до того как мы привели изменения.
     - предыдующее состояние
$ git reset HEAD

(данная команда говорим о том что: Отменить изменения файла test.txt)
вернуть предыдующее состоянее перед тем сделали коммит
вернуть предыдующее состоянее к тому состоянию на 3 последния коммита например,
для этого надо добавть флаг.

$ git reset HEAD test.txt

(после того выполняем след. команду)
$ git status

# Выполнение все изменения полностью нашего файла

$ git checkout test.txt
